# The Project - Kubernetes Configuration

This repository contains the Kubernetes manifests and kustomize configurations for The Project application (staging and production environments).

## Structure

- `kustomize/base/` - Base Kubernetes resources
- `kustomize/overlays/staging/` - Staging environment configuration
- `kustomize/overlays/production/` - Production environment configuration

## Application Code

Application source code is in the [k8s-submissions](https://github.com/duongwalker/k8s-submissions) repository.
